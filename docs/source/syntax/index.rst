Синтаксис
=====

В целом синтаксис Гадюки похож на синтаксис Python.

- Все арифметические операторы совпадают с операторами из Python 3.7

.. code-block:: python 

    a + b    a * b    a ** b    a % b
    a - b    a / b    a // b
    

- В Гадюке также используются отступы для обозначения блоков кода.
