Класс изображений
~~~~~~~~~~~~~~~~~

- Класс изображений в Гадюке образован от класса Image из модуля Pillow

Данный класс имеет 3 атрибута, доступных для пользователя:

- высота
- ширина
- размер

Высота и ширина измеряются в пикселях. Размер - это кортеж из ширины и высоты
