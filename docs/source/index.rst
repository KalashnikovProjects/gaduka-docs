Язык программирования Гадюка
===================================

**Гадюка** - язык программирования для начинающих. 
Код на гадюке очень простой, и полностью на русском языке. 
Для наглядности программирования, с помощью гадюки можно редактировать изображения.

У гадюки есть свой `сайт <https://gaduka.sytes.net/>`_ и `чат бот <https://t.me/GadukaCodeBot>`_ в телеграмме.

.. admonition:: Язык находится в разработке!
   :class: important

   На данный момент Гадюка находится в разработке. 
   
   Язык будет постепенно добаратываться!

.. admonition:: Совет для начинающих
   :class: tip

   Это техническая документация с описанием всех функций языка.
   
   Она подходит для уже знающих Python людей, но не подходит для начинающих в программировании.

.. admonition:: Удалено!
   :class: attention

   Из последней версии Гадюки были удалены команды, все они заменены на `функции <https://gaduka-docs.readthedocs.io/funcs.html>`_
