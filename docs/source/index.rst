Язык программирования Гадюка
===================================

**Гадюка** - язык программирования для начинающих. 
Код на гадюке очень простой, и полностью на русском языке. 
Для наглядности программирования, с помощью гадюки можно редактировать изображения.

У гадюки есть своё `сообщество на сайте <https://gaduka.sites.net/>`_ и `чат бот <https://t.me/GadukaCodeBot>`_ в телеграмме.

.. admonition:: Язык находится в разработке!
   :class: important

   На данный момент Гадюка находится в разработке. 
   
   Вы можете обращаться в поддержку через телеграмм бота.

   Это техническая документация с описанием всех функций языка.
